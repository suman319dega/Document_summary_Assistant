# Document Summary Assistant

A web application that allows users to upload PDF or image documents and automatically generates smart summaries using OCR and AI-based text summarization.

## Features
- Upload PDFs or Images (JPG/PNG)
- OCR using Tesseract for scanned documents
- Text extraction from PDFs
- AI-powered summaries (HuggingFace BART)
- Summary length options: Short, Medium, Long
- Responsive, modern UI
- Error handling + loading states

## Tech Stack
- Flask (Backend)
- PyPDF2 (PDF parsing)
- Tesseract OCR
- HuggingFace Transformers
- HTML/CSS/JS (Frontend)
